# TORA-DSS-2026
This collection serves as an online repository for the TORA framework introduced in the Decision Support Systems paper [Ref], including metadata, code used for empirical data collection and community rule collection, and prompt templates used with large language models.

<mark>The research project consists of THREE primary stages of implementation.</mark>

## Stage 1: Empirical Data Collection
    The project starts empirical data collection by conducting daily data collection from selective subreddits that align with your project goals and interests. Given the nature of content moderation on social media platforms and official API limitations restricted by Reddit, you would be allowed to collect up to 500 posts on each API request, depending on the filtering mechanisms you applied.
